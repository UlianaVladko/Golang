## Импортируемые пакеты
- **`fmt`** - для ввода/вывода данных
- **`slices`** - для работы со срезами и операций сортировки

## Основные функции программы

### Пользовательские функции
```go
func SortFloats64Desc(x []float64)
```
- Сортирует срез чисел по убыванию
- Использует комбинацию `slices.Sort()` и `slices.Reverse()`

### Используемые функции из стандартной библиотеки

**Из пакета `slices`:**
- `slices.Sort()` - сортировка среза по возрастанию
- `slices.Reverse()` - обратный порядок элементов
- `slices.Max()` - нахождение максимального значения
- `slices.Min()` - нахождение минимального значения

**Из пакета `fmt`:**
- `fmt.Scan()` - чтение входных данных
- `fmt.Println()` - вывод данных с переводом строки
- `fmt.Printf()` - форматированный вывод

## Конструкции языка

### Переменные и типы данных
- `var n1, n2, n3, n4, n5 float64` - объявление переменных типа float64
- `n := []float64{n1, n2, n3, n4, n5}` - создание среза float64

### Циклы
- `for _, value := range n` - range loop для итерации по срезу
- Использование `_` для игнорирования индекса

### Условные вычисления
- `average := sum / float64(len(n))` - преобразование типа int в float64

## Структуры данных
- **Срез (slice)** `[]float64` - динамический массив для хранения чисел
- **Тип float64** - числа с плавающей точкой двойной точности

## Алгоритмы
1. Чтение 5 чисел с клавиатуры
2. Создание среза из введенных чисел
3. Сортировка по убыванию (сортировка по возрастанию + реверс)
4. Поиск максимального и минимального значений
5. Вычисление среднего арифметического
6. Форматированный вывод результатов
